# tavphub

Framework **admin panel** untuk TAVP. Bikin back-office (dashboard, CRUD)
jadi cepat, nggak perlu bikin dari scratch.

## Fitur

- CRUD resources
- Table builder & form builder
- Dashboard widgets
- Sidebar navigation
- Role guard (batas akses per role)

## Cara pakai (rencana)

```
tavp module:install tavp/tavphub
```

## Status

Planning. Target `0.3.0`.
